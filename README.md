# ФИИТ.Проекты



### О проекте

Веб-сайт, с помощью которого сторонние заказчики могут подать заявку на выполнение своего проекта студентами ФИИТ.

Цель проекта: создать инструмент, который:

- Позволяет заказчику удобно и быстро оформить заявку на проект
- Систематизирует заявки и управление ими
